<h1 align="center">Spot the Gen</h1>

> An spot the difference game made with React, TypeScript, Elastic UI

Available at [spot-the-image.web.app](https://spot-the-image.web.app/)

<h2>Preview</h2> 

![Gif of the app in action](https://i.imgur.com/CkMPAtZ.gif)

Utilises: 

- React
- Typescript
- Elastic UI (Components and themes)
- Zustand (State management)
- Firebase (Hosting)
- React router

<h2></h2>

<h2>Instructions:</h2> 

<h3>Hosting</h3>

* Install Node.js

* Clone this repoistory 

* Install (e.g.): `npm install .`

* For production run:

```
npm run build
npm run start
```

* For development run:

```
npm run dev
```
